SQL SELECT และ INSERT:

SQL SELECT เป็นคำสั่งที่ใช้ในการเลือกข้อมูลจากฐานข้อมูล เพื่อให้เห็นว่าคุณสามารถเข้าใจการใช้ SELECT เพื่อเลือกข้อมูลจากตารางใด ๆ ในฐานข้อมูลของคุณได้อย่างไร
SQL INSERT เป็นคำสั่งที่ใช้ในการเพิ่มข้อมูลใหม่เข้าไปในตารางของฐานข้อมูล คุณควรอธิบายวิธีการใช้คำสั่ง INSERT เพื่อเพิ่มข้อมูลใหม่ลงในตารางและต้องระบุข้อมูลใดบ้างที่ควรระบุเมื่อทำการ INSERT เข้าไปในตาราง
SQL UPDATE และ DELETE:

SQL UPDATE เป็นคำสั่งที่ใช้ในการอัปเดตข้อมูลที่มีอยู่ในตารางของฐานข้อมูล เช่นการอัปเดตค่าของคอลัมน์ในแถวที่มีเงื่อนไขที่กำหนด
SQL DELETE เป็นคำสั่งที่ใช้ในการลบข้อมูลออกจากตารางของฐานข้อมูล เช่นการลบแถวที่มีเงื่อนไขที่กำหนด


SQL SELECT: เลือกข้อมูลจากตารางในฐานข้อมูล
SQL INSERT: เพิ่มข้อมูลใหม่เข้าไปในตารางของฐานข้อมูล

สำหรับ SQL UPDATE และ DELETE:

SQL UPDATE: อัปเดตข้อมูลที่มีอยู่ในตาราง
SQL DELETE: ลบข้อมูลออกจากตาราง
